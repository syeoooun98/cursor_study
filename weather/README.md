# 오늘의 날씨 & 미세먼지

Tkinter로 만든 실시간 날씨/미세먼지 확인 및 외출 가이드 GUI 프로그램입니다.
OpenWeatherMap API를 이용해 지역의 현재 날씨와 미세먼지(PM10)/초미세먼지(PM2.5) 농도를 조회하고,
그 결과를 바탕으로 오늘 외출이 괜찮을지, 마스크가 필요한지, 그 외 행동요령을 안내합니다.

## 기능

- 지역명(도시명, 영문/한글)을 입력하면 실시간 기온, 체감온도, 습도, 풍속을 조회합니다.
- 같은 위치의 PM10 / PM2.5 농도를 조회하고, 한국 환경부 기준(좋음/보통/나쁨/매우나쁨)으로 등급을 표시합니다.
- 날씨(강수/폭염/한파/강풍)와 미세먼지 등급을 종합해 외출 가능 여부, 마스크 착용 여부, 행동요령 목록을 안내합니다.

## 준비

`.env` 파일에 OpenWeatherMap API 키가 필요합니다.

```
weath_ApiKey = <your_api_key>
```

## 실행 방법

```bash
pip install -r requirements.txt
python gui.py
```

## 파일 구성

- `main.py` — 지오코딩, 날씨/미세먼지 조회, 외출·마스크 판단 로직 (`get_report`, `build_advice`)
- `gui.py` — Tkinter 기반 GUI 화면
